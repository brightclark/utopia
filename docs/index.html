Utopia

Utopia is a content management system built on top of Ruby. It makes creating, deploying and updating content-driven websites easy.

Middleware

The following are Rack Middleware which provide the core of Utopia.

  • Static — Serve static files efficiently.
  • Redirection — Redirect URL patterns and status codes.
  • Localization — Non-intrusive localization of resources.
  • Controller — Flexible nested controllers with efficient behaviour.
    • Controller Actions — Invoke named actions.
    • Controller Rewrite — Pattern match requests.
  • Content — XML-style template engine with dynamic markup.
  • Session — Session storage using encrypted client-side cookies.
